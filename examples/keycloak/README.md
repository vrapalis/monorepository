# KEYCLOAK EXAMPLE

## PREREQUISITE

``` 
Docker and docker compose should be installed
```

### HOW TO RUN

- navigate to docker directory
- run `` docker-compose up `` command
- after all containers are started call the frontend on `` http://localhost:4200 `` in the browser
