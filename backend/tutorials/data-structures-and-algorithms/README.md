# Data Structures And Algorithms

It is gradle based project

```
How to run default main class project: ./gradlew :tutorials:data-structures-and-algorithms:run
How to run particular main class of the project: ./gradlew :tutorials:data-structures-and-algorithms:run -P mainClass=com.vrapalis.www.tutorials.datastructuresandalgorithms.datastructures.stack.RunArrayBoundStack
How to test project: ./gradlew :tutorials:data-structures-and-algorithms:test
```
